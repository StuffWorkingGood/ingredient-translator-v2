# Contributing

Thanks for your interest in contributing to Ingredient Translator!

## How to Contribute
- Fork the repository
- Create a new branch for your change
- Make your updates
- Submit a pull request describing what you changed and why

## Guidelines
- Keep changes focused and minimal
- Follow the existing structure and style
- Test changes before submitting

## Questions or Ideas
If you have a suggestion or improvement idea, feel free to open an issue.
